# Imágenes del Recorrido Virtual 360°

Fotos panorámicas/360° para el recorrido interactivo.

## Archivos esperados:

1. **entrada.jpg** - Entrada principal
2. **piscina.jpg** - Piscina privada
3. **sala.jpg** - Sala de estar
4. **cocina.jpg** - Cocina equipada
5. **habitacion.jpg** - Habitación principal
6. **bbq.jpg** - Zona BBQ

## Recomendaciones:

- Formato: JPG o WebP (optimizado)
- Resolución: 1920x1080 o superior
- **Idealmente fotos 360° de cada zona** (para recorrido interactivo)
- Si no tienes 360°, usa fotos panorámicas o buenas vistas de cada espacio
