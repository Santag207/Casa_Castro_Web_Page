# Imágenes - Página Principal (Home)

Coloca aquí las imágenes de la página de inicio.

- **hero.jpg** - Imagen de fondo principal (grande, hero section)
- **piscina-feature.jpg** - Imagen de la sección "Piscina privada y zonas verdes"
