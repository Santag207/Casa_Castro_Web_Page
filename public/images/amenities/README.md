# Imágenes de Amenidades

Imágenes de las características principales de la casa.

## Archivos esperados:

1. **piscina.jpg** - Piscina privada
2. **bbq.jpg** - Zona BBQ/parrilla
3. **casa-cerrada.jpg** - Fachada o entrada (privacidad)
4. **cocina.jpg** - Cocina equipada
5. **sala.jpg** - Sala de estar
6. **jardin.jpg** - Zonas verdes/jardín
7. **aire-acondicionado.jpg** - Habitación con A/A o termostato

## Recomendaciones:

- Formato: JPG o WebP (optimizadas)
- Resolución: 800x600
- Una foto clara de cada amenidad principal
